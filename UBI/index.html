UBI Usability Evaluation Report

Problem Statement

The objective is to evaluate the usability of the Union Bank of India's internet banking website using Nielsen's 10 Usability Heuristics. The goal is to identify usability problems and provide recommendations for improvement.

Bank Name

Union Bank of India (UBI) - Internet Banking

Goals

  • Assess the usability of the UBI internet banking site.
  • Identify usability issues based on Nielsen's heuristics.
  • Measure the efficiency and effectiveness of the site.
  • Provide actionable recommendations to improve the siteâ€™s usability.

Methods Adopted for Validation

Heuristic Evaluation

Conduct a heuristic evaluation using Nielsen's 10 Usability Heuristics.

Usability Testing

Observe users performing specific tasks on the site and measure time invested and efficiency.

Surveys/Questionnaires

Collect user feedback on their experience with the site.

Findings

Results for Time Invested and Efficiency

Measure the time taken by users to complete common banking tasks (e.g., logging in, checking account balance, transferring funds).

Results by Number of Usability Problems

Count the number of usability issues identified in each heuristic category.

Results by Severity

Categorize the severity of each usability issue (e.g., minor, major, critical).

Results by Degree of Perception

Evaluate how users perceive the usability of the site based on surveys/questionnaires.

Obstacles Faced and How to Overcome Them

Obstacle: Difficulty in finding users for testing

Solution: Reach out to friends, family, and colleagues who use UBI internet banking.

Obstacle: Limited access to certain features due to security concerns

Solution: Use available demo accounts or coordinate with the bank for temporary access.

Tools Used for Evaluating Interface and Reporting

  • Document Tools: MS Office for documentation.
  • Wireframe Tools: MockFlow for creating wireframes.
  • Browser: Google Chrome for accessing the UBI internet banking site.

Recommendations for Improvement

Based on the findings from the heuristic evaluation and usability testing, provide specific recommendations to improve the site's usability. For example:

  • Improve the visibility of system status by providing clear feedback to users.
  • Ensure a match between the system and the real world by using familiar terminology.
  • Enhance user control and freedom by allowing easy navigation and undo options.
